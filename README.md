# Dashdojo ui-components

## Project setup

```bash
npm install
```

### Run StoryBook

```bash
npm run storybook
```

### Test (all files src/)

```bash
npm test
```

### Test single conponent

```bash
npm run test:single src/packages/block/block.test.js
```

### Release

```bash
npm run release
```

### Lints and fixes files

```bash
npm run lint
```

### Reinstall node_modules

```bash
npm run reinstall
```
