# @dashdojodev/ui-components

### Package install

```bash
npm install @dashdojodev/ui-components -S
```

### Project setup

```bash
npm install
```

### Run StoryBook

```bash
npm run storybook
```

### Test (all files src/)

```bash
npm test
```

### Test single component

```bash
npm run test:single src/packages/block/block.test.js
```

### Release

```bash
npm run release
```

### Lint and fix files

```bash
npm run lint
```

### Reinstall node_modules

```bash
npm run reinstall
```
